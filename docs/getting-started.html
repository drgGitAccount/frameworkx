Getting Started

An overview of VisualUI, how to download and use it, and some basic templates and examples.

What is VisualUI?

VisualUI is a modern, production-ready CSS framework that combines the best features of Bootstrap and Tailwind CSS. It provides:

  • Pre-built components for rapid development
  • Comprehensive utility classes for custom designs
  • Responsive grid system
  • Modern CSS variables for easy theming
  • Lightweight JavaScript components
  • Mobile-first approach

Download

Get started with VisualUI in your project by including the pre-built CSS and JS files.

CDN

The quickest way to get started is to use the CDN:

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/VisualUI@1.0.0/dist/VisualUI.css" rel="stylesheet">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/VisualUI@1.0.0/dist/VisualUI.js"></script>

Package Managers

Install VisualUI using your favorite package manager:

# NPM
npm install VisualUI

# Yarn
yarn add VisualUI

# Bower
bower install VisualUI

Browser Support

VisualUI supports the latest, stable releases of all major browsers:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)
Note: Internet Explorer is not supported.

Customization

VisualUI is built with CSS variables, making it easy to customize the appearance:

Override Variables

Create your own CSS file and override the default variables:

/* custom.css */
:root {
  --fw-color-primary: #your-color;
  --fw-color-secondary: #your-color;
  --fw-font-family-base: 'Your Font', sans-serif;
  --fw-border-radius: 8px;
}

Custom Build

For more advanced customization, you can modify the source files and rebuild:

# Install dependencies
npm install

# Build CSS
npm run build:css

# Build JS
npm run build:js

Build Tools

VisualUI can be integrated with various build tools:

Webpack

// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};

Sass/SCSS

While VisualUI uses pure CSS, you can still import it into your Sass files:

// main.scss
@import 'VisualUI/src/VisualUI.css';

// Your custom styles
.custom-component {
  // Your styles here
}

PostCSS

// postcss.config.js
module.exports = {
  plugins: [
    require('postcss-import'),
    require('autoprefixer'),
    require('cssnano')
  ]
};

Basic Template

Here's a basic HTML template to get you started:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My VisualUI App</title>
    <link href="path/to/VisualUI.css" rel="stylesheet">
</head>
<body>
    <div class="fw-container">
        <div class="fw-row">
            <div class="fw-col-12">
                <h1>Hello, VisualUI!</h1>
                <p>Welcome to your new app.</p>
            </div>
        </div>
    </div>
    
    <script src="path/to/VisualUI.js"></script>
</body>
</html>
