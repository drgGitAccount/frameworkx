Getting Started

Learn how to install FrameworkX and start building modern, responsive websites.

What is FrameworkX?

FrameworkX is a modern, production-ready CSS framework that combines the best features of Bootstrap and Tailwind CSS. It provides:

  • Pre-built components - Buttons, cards, forms, modals, and more for rapid development
  • Utility classes - Comprehensive utilities for custom designs without writing CSS
  • Responsive grid system - Mobile-first layout that adapts to any screen size
  • CSS variables - Easy theming and customization with modern CSS
  • Lightweight JavaScript - Optional interactive components like modals and dropdowns
  • Optimized for performance - Minimal footprint with no external dependencies

Download

Get started with FrameworkX in your project by including the pre-built CSS and JS files.

CDN (Recommended for Quick Start)

The quickest way to get started is to use the CDN. No installation required:

<!-- Add this in your <head> -->
<link href="https://cdn.jsdelivr.net/npm/frameworkx@1.0.0/dist/frameworkx.css" rel="stylesheet">

<!-- Add this before closing </body> -->
<script src="https://cdn.jsdelivr.net/npm/frameworkx@1.0.0/dist/frameworkx.js"></script>

NPM (Recommended for Projects)

Install FrameworkX via npm for better version control and integration:

npm install frameworkx

Download Files

Download the compiled and minified versions:

Browser Support

FrameworkX supports the latest, stable releases of all major browsers:

  • Chrome (latest)
  • Firefox (latest)
  • Safari (latest)
  • Edge (latest)
  • Opera (latest)
Note: Internet Explorer is not supported.

Customization

FrameworkX is built with CSS variables, making it easy to customize the appearance:

Override Variables

Create your own CSS file and override the default variables:

/* custom.css */
:root {
  --fw-color-primary: #your-color;
  --fw-color-secondary: #your-color;
  --fw-font-family-base: 'Your Font', sans-serif;
  --fw-border-radius: 8px;
}

Custom Build

For more advanced customization, you can modify the source files and rebuild:

# Install dependencies
npm install

# Build CSS
npm run build:css

# Build JS
npm run build:js

Build Tools

FrameworkX can be integrated with various build tools:

Webpack

// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};

Sass/SCSS

While VisualUI uses pure CSS, you can still import it into your Sass files:

// main.scss
@import 'VisualUI/src/VisualUI.css';

// Your custom styles
.custom-component {
  // Your styles here
}

PostCSS

// postcss.config.js
module.exports = {
  plugins: [
    require('postcss-import'),
    require('autoprefixer'),
    require('cssnano')
  ]
};

Basic Template

Here's a basic HTML template to get you started:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My VisualUI App</title>
    <link href="path/to/VisualUI.css" rel="stylesheet">
</head>
<body>
    <div class="fw-container">
        <div class="fw-row">
            <div class="fw-col-12">
                <h1>Hello, VisualUI!</h1>
                <p>Welcome to your new app.</p>
            </div>
        </div>
    </div>
    
    <script src="path/to/VisualUI.js"></script>
</body>
</html>
