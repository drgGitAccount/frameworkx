Alerts

Display contextual feedback messages for user actions with flexible alert components.

Examples

Alerts come in 8 different colors to convey different message types:

Primary Alert: This is a primary alert message for important information.
Success Alert: Great! Your action was completed successfully.
Danger Alert: There was an error. Please try again.
Warning Alert: Please review this information carefully.
Info Alert: Here's some helpful information for you.

Code Example

<div class="fw-alert fw-alert-success" role="alert">
    <strong>Success!</strong> Your action was completed successfully.
</div>

Grid System

Build responsive layouts with containers, rows, and 12-column spans. Gutters are handled automatically by rows and columns.

Classes

Use these core layout classes throughout your layouts:

  • fw-container, fw-container-fluid, fw-container-sm, fw-container-md, fw-container-lg, fw-container-xl
  • fw-row
  • fw-col, fw-col-auto, fw-col-1 through fw-col-12
  • fw-col-sm, fw-col-md, fw-col-lg, fw-col-xl
  • fw-col-sm-*, fw-col-md-*, fw-col-lg-*, fw-col-xl-*
