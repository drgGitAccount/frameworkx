Get Started in Minutes

Simple installation and easy setup for any project.

Using CDN (Quickest)

Add FrameworkX to your project with a simple link tag:

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/frameworkx@1.0.0/dist/frameworkx.css" rel="stylesheet">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/frameworkx@1.0.0/dist/frameworkx.js"></script>

Basic Usage

Start building with FrameworkX components:

<div class="fw-container">
  <div class="fw-row">
    <div class="fw-col-md-6">
      <div class="fw-card">
        <div class="fw-card-body">
          <h3 class="fw-card-title">Welcome!</h3>
          <p class="fw-card-text">Your first FrameworkX component.</p>
          <button class="fw-btn fw-btn-primary">Get Started</button>
        </div>
      </div>
    </div>
  </div>
</div>
