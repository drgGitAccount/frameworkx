VisualUI

A production-ready CSS framework that combines the best of Bootstrap and Tailwind CSS. Modern, flexible, and easy to extend.

Get Started View Examples

Why VisualUI?

Production Ready

Built with production in mind. Clean, optimized, and thoroughly tested code that you can trust.

Utility First

Comprehensive utility classes for rapid development. Build custom designs without writing custom CSS.

Component Based

Pre-built components that work out of the box. Buttons, cards, forms, modals, and more.

Quick Start

Installation

Add VisualUI to your project with a simple link tag:

<link href="https://cdn.jsdelivr.net/npm/VisualUI@1.0.0/dist/VisualUI.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/VisualUI@1.0.0/dist/VisualUI.js"></script>

Basic Usage

Start building with VisualUI components:

<div class="fw-container">
  <div class="fw-row">
    <div class="fw-col-md-6">
      <h1>Hello VisualUI!</h1>
      <button class="fw-btn fw-btn-primary">Click me</button>
    </div>
    <div class="fw-col-md-6">
      <div class="fw-card">
        <div class="fw-card-body">
          <h5 class="fw-card-title">Card Title</h5>
          <p class="fw-card-text">Some quick example text.</p>
        </div>
      </div>
    </div>
  </div>
</div>
