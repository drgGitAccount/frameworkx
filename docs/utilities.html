Spacing Utilities

FrameworkX includes a wide range of shorthand responsive margin and padding utility classes to modify an element's appearance.

How it works

Assign responsive-friendly margin or padding values to an element or a subset of its sides with shorthand classes. Includes support for individual properties, all properties, and vertical and horizontal properties.

Notation

Spacing utilities that apply to all breakpoints, from xs to xl, have no breakpoint abbreviation in them. This is because those classes are applied from min-width: 0; and up, and thus are not bound by a viewport.

Display Utilities

Quickly and responsively toggle the display value of components and more with our display utilities.

How it works

Change the display value of elements with our responsive display utility classes. We purposely support only a subset of all possible values for display. Classes can be combined for various effects as you need.

Examples

d-block
d-inline
d-inline-block
d-none (hidden)

Hiding elements

For faster mobile-friendly development, use responsive display classes for showing and hiding elements by device. Avoid creating entirely different versions of the same site, instead hide elements responsively for each screen size.

Code

<div class="d-inline">Inline</div>
<div class="d-inline-block">Inline block</div>
<div class="d-block">Block</div>
<div class="d-grid">Grid</div>
<div class="d-table">Table</div>
<div class="d-table-cell">Table cell</div>
<div class="d-flex">Flex</div>
<div class="d-none">Hidden</div>

Position Utilities

Use these shorthand utilities for quickly configuring the position of an element.

Common positioning values

Quick positioning classes are available, though they are not responsive.

Static
Static flow Sibling
Elements remain in normal document flow.
Relative
Anchor Relative
Relative offsets move the element without leaving flow.
Absolute
Absolute
Absolutely positioned inside a relative parent.
Fixed
Viewport Fixed
Pinned to the viewport while you scroll.

Center elements

In addition, you can also center the elements with the transform utility class.

Centered

Sticky elements

Sticky positioned elements stay within the flow until they reach the top of their scroll container.

Sticky header

Scroll inside this container to see the sticky element hold its position at the top.

Additional content line to create scrollable space and demonstrate the effect.

Keep scrolling to confirm the sticky behavior remains consistent.

Shadow Utilities

Add or remove shadows to elements with box-shadow utilities.

Examples

While shadows on components are disabled by default in FrameworkX and can be enabled via CSS variables, you can also quickly add or remove shadows with our box-shadow utility classes. Includes support for .shadow-none and three default sizes (which have associated variables to match).

No shadow
Small shadow
Regular shadow
Larger shadow

Code

<div class="shadow-sm">Small shadow</div>
<div class="shadow">Regular shadow</div>
<div class="shadow-lg">Larger shadow</div>
